### 公司标签

**FAB / AI基础设施 / 半导体设备 / 先进封装 / GPU/TPU/ASIC代工**

### 核心业务卡位

台积电是全球最大、技术最先进的晶圆代工厂，掌握从3nm到2nm及以下先进制程的绝对主导权，为英伟达、苹果、AMD等全球顶级芯片设计公司提供制造服务，同时通过CoWoS先进封装深度卡位AI算力供应链。

## 模块2：竞争与护城河

### 直接竞争对手


| 竞争对手                      | 技术节点              | 竞争态势                                      |
| ------------------------- | ----------------- | ----------------------------------------- |
| **三星电子（Samsung Foundry）** | 3nm GAA已量产，2nm开发中 | 技术代差约1-1.5年，良率和能效落后台积电，正通过降价争夺中低端先进工艺订单   |
| **英特尔（Intel Foundry）**    | 18A（等效1.8nm）推进中   | IDM 2.0转型困难，先进代工市场份额极低，技术 roadmap 执行不确定性高 |
| **中芯国际（SMIC）**            | 7nm（成熟工艺为主）       | 受出口管制限制，主要聚焦成熟制程和国内客户，对台积电直接影响有限          |


### 技术代差评估

- **3nm节点**：台积电N3E/N3P已大规模量产，良率和性能全面领先三星；苹果、英伟达、AMD核心产品均锁定台积电
- **2nm节点**：台积电N2预计2025年量产，N2P/A16在2026年爬坡；三星2nm GAA量产时间滞后
- **先进封装**：台积电CoWoS是AI加速器（如NVIDIA Blackwell GPU）的主流2.5D解决方案，2024年占据22%市场份额（除索尼外最大供应商），技术定义者地位稳固

### 客户自研替代风险

- **风险较低**：英伟达、苹果、AMD等核心客户虽具备芯片设计能力，但均不拥有先进晶圆厂；自建晶圆厂资本投入极高（单座先进晶圆厂超200亿美元）且技术积累需10年以上
- **潜在风险点**：部分客户（如苹果）可能在成熟制程上分散供应链，但在3nm及以下节点台积电仍不可替代
- **地缘政治因素**：美国推动产能本土化，台积电亚利桑那厂投产可在一定程度上缓解客户对地缘风险的担忧

## 模块3：地缘政治与监管

### 出口管制/实体清单

- **2025年1月**：台积电向中国大陆IC设计公司发出通知，自2025年1月31日起，16/14nm及以下产品未在BIS白名单中的"approved OSAT"进行封装且未收到认证签署副本的，发货将被暂停 
- **2025年12月**：特朗普政府允许英伟达向中国销售H200芯片，为出口管制的重大松动 
- **2026年6月**：美国商务部BIS发布澄清通知，确认AI芯片出口限制适用于总部或母公司位于中国的所有企业（包括境外子公司），堵住潜在漏洞

### 关税/供应链重组

- **2026年1月美台贸易协议**：台湾承诺为其芯片和科技公司在美国扩大产能提供2,500亿美元信贷担保；美国商务部长表示目标是将40%的台湾半导体供应链带到美国 
- **台积电美国投资**：已承诺在美投资1,650亿美元建设晶圆厂和研发中心 
- **N-2规则**：台湾当局限制台积电海外工厂技术至少落后本土两代，最先进制程仍留在台湾 
- **关税影响**：台积电管理层在法说会中表示2025年全年因关税相关影响预计在5,000万-8,000万欧元之间（对TSM直接影响较小，主要影响客户成本结构）

### 反垄断调查

- 台积电作为全球晶圆代工绝对龙头，在多个国家和地区面临反垄断关注，但目前未遭遇重大拆分或罚款风险

### AI监管

- 美国对华AI芯片出口管制持续收紧，直接影响台积电中国客户的先进制程订单；荷兰、日本设备管制影响全球先进产能扩张节奏

## 模块4：技术路线图

### 下一代产品时间线


| 时间节点     | 技术节点             | 关键特征                                    |
| -------- | ---------------- | --------------------------------------- |
| 2025年    | N3E/N3P量产，N2开始量产 | 3nm家族全面铺开，2nm正式进入量产阶段                   |
| 2026年    | N2P、A16、N3A      | N2P性能提升5-10%，A16面向高端移动和AI加速器，ASP较N2高20% |
| 2027年    | N2X、A14          | 2nm家族扩展，1.4nm技术迭代启动                     |
| 2028年及以后 | A14量产            | 继续推进摩尔定律，探索GAAFET之后的新器件结构               |


### 关键产品良率/产能释放

- **亚利桑那Fab 21**：良率92%，超过台湾本土工厂约4个百分点，第一年盈利5.14亿美元 
- **N2制程**：产能爬坡顺利，客户包括苹果、英伟达等头部厂商
- **CoWoS先进封装**：产能持续满载，溢出订单加速流向日月光、安靠等传统封测巨头

### 软件生态绑定

- 台积电通过**开放创新平台（OIP）**与EDA工具商（Cadence、Synopsys）、IP核供应商深度绑定
- 与英伟达、苹果、AMD等客户建立长期战略合作关系，共同定义先进制程设计规则和封装标准
