## 模块2：竞争与护城河

### 2.1 直接竞争对手


| 竞争对手               | 核心产品                   | 威胁等级      |
| ------------------ | ---------------------- | --------- |
| **Apple (AAPL)**   | 自研C1/C2基带芯片、A系列/M系列SoC | **极高**    |
| **MediaTek**       | 天玑系列手机SoC，中低端市场        | 高         |
| **Samsung LSI**    | Exynos系列SoC            | 中         |
| **NVIDIA (NVDA)**  | 数据中心GPU                | 中（数据中心领域） |
| **Marvell (MRVL)** | 数据中心互连/DPU             | 中         |
| **Intel (INTC)**   | PC处理器、基带（已退出）          | 低         |


### 2.2 技术代差评估

- **智能手机SoC**：高通骁龙8系仍是安卓阵营旗舰首选，但MediaTek天玑在高端市场（如vivo、OPPO旗舰机）份额持续提升，价格竞争加剧
- **Apple自研威胁**：Apple已在iPhone 16e中部署自研C1基带，直接威胁高通最大客户。市场担忧Apple将在更多机型中替代高通基带，造成收入悬崖
- **PC芯片**：Snapdragon X Elite进入Windows on Arm市场，但渗透速度慢于市场预期，面临Intel和AMD的激烈竞争
- **汽车芯片**：汽车业务收入连续两季超10亿美元（Q1 FY26 $1.1B，+15% YoY），是重要增长极
- **数据中心**：新发布Dragonfly品牌，定位推理工作负载，但相比NVIDIA生态尚处早期

### 2.3 客户自研替代风险

- **Apple风险**：这是高通最大的单一客户风险。Apple C1/C2基带若扩展至更多iPhone机型，将直接冲击高通QCT收入（iPhone贡献高通收入的约20-25%）
- **OEM自研**：Samsung、小米等也在探索自研芯片，但短期内难以替代高通旗舰SoC
- **时间线**：Apple C1已在iPhone 16e上使用，预计2026-2027年将继续扩展，高通的Apple收入下降是确定性趋势，关键问题是速度

## 模块3：地缘政治与监管

### 3.1 出口管制/实体清单

- **中国手机市场需求疲软**：中国智能手机市场疲软叠加MediaTek中低端竞争，挤压高通在华收入
- **华为禁令**：华为受美国制裁后无法采购高通5G芯片，高通失去一个重要客户（虽然华为已部分恢复4G芯片采购）
- **出口管制对数据中心的影响**：若高通Dragonfly数据中心芯片涉及先进制程，可能受对华出口管制约束

### 3.2 关税/供应链重组

- **关税影响**：高通芯片主要在亚洲生产（TSMC/Samsung代工），若美国对中国/台湾关税升级将影响成本
- **供应链分散**：Fabless模式对供应链的依赖主要是代工厂，高通本身不直接受制造业关税影响，但终端产品（手机）关税会影响客户需求

### 3.3 反垄断调查

- **历史案例**：高通曾在中国、韩国、欧盟、美国面临反垄断调查和巨额罚款（与专利授权费率和捆绑销售有关）
- **当前状态**：近年来反垄断风险有所下降，但专利授权模式（QTL业务）仍是监管关注焦点
- **与Apple的诉讼**：2019年达成和解，但Apple自研基带趋势使双方关系持续紧张

### 3.4 AI监管

- **端侧AI监管**：欧盟AI法案对高风险AI系统的定义可能影响智能手机AI功能的合规要求
- **汽车AI**：智能驾驶相关的AI监管趋严，可能影响高通Snapdragon Ride平台的推广节奏

## 模块4：技术路线图

### 4.1 下一代产品时间线

- **Snapdragon 8 Gen 5 / Gen 6**：预计2025-2026年发布，继续强化NPU AI算力
- **Snapdragon X Elite迭代**：下一代PC芯片，目标提升Windows on Arm体验和市场接受度
- **Dragonfly数据中心品牌**：2026年Computex正式发布，定位数据中心推理市场，已与超大规模云厂商和全球伙伴合作部署
- **6G研发**：CEO将6G定位为首个AI原生无线网络，连接分布式混合智能
- **Nuvia自研CPU架构**：用于PC和数据中心，减少对外部CPU IP的依赖

### 4.2 关键产品良率/产能释放

- **手机芯片**：旗舰SoC在TSMC 3nm/4nm节点生产，良率稳定，产能充足
- **PC芯片**：Snapdragon X Elite产能不是问题，瓶颈在于软件生态兼容性和消费者接受度
- **汽车芯片**：车规级认证周期长，收入确认滞后于设计 win
- **数据中心芯片**：Dragonfly处于早期部署阶段，2026-2027年将是产能和客户验证的关键期

### 4.3 软件生态绑定

- **Android生态**：高通与Google Android深度绑定，旗舰SoC获得首发优化
- **Windows on Arm**：与Microsoft合作推动Arm架构PC，但软件兼容性仍落后于x86
- **汽车软件栈**：Snapdragon Ride平台包含完整的自动驾驶软件栈，与客户深度绑定
- **AI开发工具**：Qualcomm AI Stack支持主流框架（TensorFlow、PyTorch），降低开发者迁移成本

### 8.1 主要厂商模型发布节奏

参见Broadcom (AVGO) 文档模块8。

### 8.2 对Qualcomm的特别影响

- **Agentic AI（自主智能体）**：Qualcomm CEO在Computex 2026明确将2026年定义为"Agent之年"，AI从提示交互转向自主规划、推理和行动
- **端侧推理需求**：Agentic AI需要在终端设备（手机、PC、汽车、IoT）上运行复杂推理任务，对NPU算力、内存带宽和能效提出极高要求
- **Qualcomm定位**：CEO强调公司可从"亚2毫瓦"（耳机等）到"千瓦级"（数据中心）覆盖全算力连续体
- **模型小型化趋势**：端侧AI推动模型蒸馏和量化，使得百亿参数级模型可在手机NPU上运行，直接利好高通骁龙平台的AI性能卖点
- **6G与AI原生网络**：CEO将6G定位为首个AI原生无线网络，连接分布式混合智能，这可能重塑高通的通信专利组合价值
