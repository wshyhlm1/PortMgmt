SK Telecom是韩国最大的移动通信运营商，正从传统电信运营商向"AI公司"战略转型，通过5G网络基础设施、AI数据中心、云服务和产业数字化解决方案，定位东亚AI生态的关键基础设施提供商。

## 模块2：竞争与护城河

### 2.1 直接竞争对手


| 竞争对手                      | 核心优势              | 威胁等级   |
| ------------------------- | ----------------- | ------ |
| **KT Corporation**        | 韩国第二大运营商，固网+企业业务强 | 高      |
| **LG U+**                 | 价格竞争激进，内容/媒体差异化   | 高      |
| **Amazon AWS / MS Azure** | 云服务的全球竞争者         | 中（企业云） |
| **Naver / Kakao**         | 本土互联网巨头，AI和内容生态   | 中      |


### 2.2 技术代差评估

- **5G网络**：SK Telecom在韩国5G部署上领先，但KT和LG U+差距不大，网络质量同质化
- **AI转型**：SKT是韩国电信运营商中AI转型最激进的一家，成立了独立的AI子公司和DXC（数字体验中心）业务，但相比全球科技巨头仍处于早期阶段
- **护城河**：牌照壁垒（电信运营牌照）+ 品牌认知 + 企业客户关系；但技术护城河有限，面临互联网公司跨界竞争

### 2.3 客户自研替代风险

- **企业客户**：大型企业可能直接采购AWS/Azure服务，绕过SKT的企业云/ICT解决方案
- **消费者**：用户对运营商忠诚度较低，价格敏感度高，携号转网便利
- **内容替代**：Wavve等本土OTT面临Netflix、Disney+等全球平台竞争

## 模块3：地缘政治与监管

### 3.1 出口管制/实体清单

- **AI芯片采购**：韩国未受美国对华AI芯片出口管制的直接限制，SKT可正常采购NVIDIA GPU等先进AI芯片
- **中美科技战溢出效应**：若中美关系进一步紧张，韩国作为美国盟友可能面临选边压力，影响在华业务（SKT在华业务有限）

### 3.2 关税/供应链重组

- **直接关税影响有限**：电信服务为本地业务，不受国际贸易关税直接影响
- **设备供应链**：5G基站设备部分依赖华为（历史）和爱立信/诺基亚，供应链相对稳定

### 3.3 反垄断调查

- **韩国电信监管**：韩国政府长期关注电信市场寡头垄断问题，KT/SKT/LG U+三家主导市场，面临资费管制和互联互通监管
- **频谱拍卖**：政府通过频谱拍卖和价格管制平衡运营商利润

### 3.4 AI监管

- **韩国AI基本法**：韩国正制定AI相关法规，SKT作为AI服务提供商需关注合规成本
- **数据隐私**：韩国个人信息保护法（PIPA）严格，影响AI训练和用户数据分析
