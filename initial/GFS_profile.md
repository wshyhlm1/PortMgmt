**FAB**、**AI基础设施**、**光互联**、**智能驾驶**、**半导体设备**

### 1.3 核心业务卡位

GlobalFoundries（格芯）是全球第三大晶圆代工厂，专注于差异化特色工艺（非最先进制程竞争），在硅光子、FD-SOI、RF-SOI、SiGe、BCD、GaN等特种工艺领域具有领先地位，受益于AI数据中心互联（硅光子）、汽车电子、物联网和半导体供应链本土化趋势。

## 模块2：竞争与护城河

### 2.1 竞争格局

GlobalFoundries在晶圆代工市场面临的主要竞争对手：


| 竞争对手                | 定位            | 与GFS关系          |
| ------------------- | ------------- | --------------- |
| 台积电(TSMC)           | 全球代工龙头，先进制程主导 | 差异化竞争，GFS聚焦特色工艺 |
| 三星代工                | 存储+逻辑代工       | 在成熟制程领域竞争       |
| 中芯国际(SMIC)          | 中国大陆代工龙头      | 地缘政治推动客户转向GFS   |
| 联电(UMC)             | 成熟制程代工        | 直接竞争            |
| Tower Semiconductor | 特种工艺代工        | 领域重叠            |
| Intel Foundry       | 新进入者          | 潜在竞争者           |


### 2.2 GlobalFoundries的竞争优势

1. **差异化技术平台**：不追逐最先进制程（如3nm/2nm），而是在硅光子、FD-SOI、RF-SOI、SiGe、BCD等特色工艺建立领先地位
2. **全球多元化制造布局**：美国、欧洲、亚洲三地布局，是唯一能提供三大洲制造选择的代工厂。2025年因制造布局优势获得的新设计订单预计生命周期收入超$30亿
3. **硅光子领先地位**：深耕硅光子十余年，通过收购AMF和InfiniLink强化平台。2025年收入接近翻倍，目标2028年底年化收入达$10亿，2030年超$10亿
4. **汽车电子增长**：汽车收入从2020年占比2%提升至2025年21%，智能传感器和网络收入2025年同比增超三倍
5. **政府支持和本土化趋势**：受益于半导体供应链回流（onshoring），客户明确要求"非中国、非台湾"采购
6. **高客户粘性**：2025年超500个设计定案，其中超95%为独家承接

### 2.3 护城河评估

- **技术差异化（强）**：在多个特色工艺领域建立深厚积累和客户认证
- **制造布局（极强）**：三大洲布局在当前地缘政治环境下成为核心竞争优势
- **客户关系和认证（强）**：汽车等长周期行业客户转换成本高
- **规模效应（中）**：在特色工艺领域具备规模，但总体规模小于台积电

**风险点**：

- 智能手机业务仍占39%，该市场增长放缓且价格竞争激烈
- 先进制程能力有限，无法参与最前沿AI芯片制造
- 产能利用率（85%）仍有提升空间
- Intel Foundry未来可能加大竞争

## 模块3：地缘政治与监管

### 3.1 半导体供应链本土化（Onshoring）

- **核心受益者**：地缘政治紧张、关税和出口管制推动企业将半导体供应链迁回本土
- **客户要求**：越来越多公司明确要求"非中国、非台湾"采购渠道
- **美国供应链战略**：美国企业将美国作为长期供应链布局核心
- **GF优势**：灵活覆盖美国、欧洲、亚洲，满足客户多元化采购需求

### 3.2 政府补贴与CHIPS法案

- **美国CHIPS法案**：GFS获得补贴支持美国工厂扩建
- **纽约州Green CHIPS**：额外州级补贴
- **德国补贴**：支持Dresden工厂扩建
- **新加坡**：政府支持硅光子产能扩张
- **风险**：若补贴延迟或取消，可能影响扩张计划

### 3.3 出口管制

- 目前GFS的工艺节点（12nm及以上为主）不在最先进制程出口管制范围内
- 需关注美国是否扩大对华成熟制程出口限制
- 硅光子、GaN等新兴领域可能成为管制焦点

### 3.4 关税影响

- 晶圆制造全球化程度高，关税对直接影响有限
- 但设备和材料进口可能受关税影响
- 管理层表示2026年定价环境整体向好，部分行业参与者已开始提价

## 模块7：AI Capex专项

### 7.1 GFS与AI Capex的关系

GlobalFoundries并非AI Capex的直接支出方，而是**AI Capex间接受益方**，主要通过以下路径：

1. **AI数据中心光互联**：硅光子芯片是AI数据中心高速互联的核心组件
2. **AI芯片电源管理**：BCD和GaN工艺用于AI芯片电源管理
3. **边缘AI/物理AI**：FDX和SiGe工艺支持边缘AI推理和传感器
4. **先进封装**：为AI芯片提供封装解决方案

### 7.2 AI驱动的高增长走廊


| 业务领域        | 2025表现   | 2026预期        | AI Capex关联      |
| ----------- | -------- | ------------- | --------------- |
| 硅光子         | 收入接近翻倍   | 预计再次翻倍        | 直接受益于数据中心光互联投资  |
| 通信基础设施与数据中心 | 占收入11%   | 预计同比增长超30%    | AI数据中心网络和SATCOM |
| 汽车智能传感器     | 收入同比增超3倍 | 持续高增长         | 自动驾驶/物理AI传感器    |
| GaN电源管理     | 开发中      | 预计2026-2027量产 | AI数据中心电源效率      |


### 7.3 客户AI Capex映射

- **数据中心客户**：通过硅光子、SiGe、GaN等工艺受益于云服务商和AI公司的基础设施投资
- **网络设备商**：如Ciena等使用GFS硅光子芯片制造光网络设备
- **汽车客户**：受益于智能驾驶和AI驱动的汽车电子升级

### 7.4 产能投资与AI需求

- 2026年CAPEX重点投向硅光子、FDX、SiGe等AI相关高增长领域
- 这些领域产能已超订，需要扩产满足需求
- 客户预付款和政府补贴为CAPEX提供资金支持

## 模块8：AI模型发布专项

### 8.1 AI模型对芯片制造的需求

AI模型的演进推动对以下芯片的需求：

- **训练芯片**：虽然GFS不生产最先进GPU，但为AI芯片提供电源管理、I/O和封装
- **推理芯片**：FDX平台适合低功耗边缘AI推理
- **光互联芯片**：硅光子芯片支持AI集群高速互联
- **传感器芯片**：物理AI需要大量智能传感器

### 8.3 物理AI（Physical AI）机遇

GFS将"Physical AI"视为重大趋势：

- **定义**：AI从虚拟世界（聊天、生成内容）进入物理世界（机器人、自动驾驶、工业自动化）
- **对GFS的影响**：物理AI需要大量边缘计算芯片、传感器、射频芯片和电源管理芯片
- **GFS定位**：FDX、SiGe、BCD、RF-SOI等工艺完美匹配物理AI芯片需求
- **收购Synopsys ARC IP**：强化嵌入式AI处理器能力，直接服务物理AI市场

### 8.4 硅光子与AI模型训练/推理

- 随着AI模型规模扩大，分布式训练集群需要更高带宽光互联
- 从可插拔光模块向共封装光学(CPO)演进
- GFS作为全球最大的纯硅光子代工厂，处于核心位置
- 与Corning等合作伙伴共同推进CPO生态
